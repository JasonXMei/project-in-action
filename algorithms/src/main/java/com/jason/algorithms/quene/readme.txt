与堆栈类似，队列是另一种线性数据结构，以顺序方式存储元素。堆栈和队列之间唯一的显着区别是，
队列不是使用 LIFO 方法，而是应用 FIFO 方法，这是 First in First Out（先入先出）的缩写。

队列的完美现实例子：
一列人在售票亭等候。如果有新人来，他们是从末尾加入队列，而不是在开头——站在前面的人将先买到票然后离开队列。

队列的基本操作：
Enqueue() —— 向队列末尾插入元素
Dequeue() —— 从队列头部移除元素
isEmpty() —— 如果队列为空，则返回 true
Top() —— 返回队列的第一个元素

常问的队列面试问题：
使用队列来实现堆栈
颠倒队列中前 k 个元素的顺序
使用队列生成从 1 到 n 的二进制数
